# Contributing

Thank you for helping improve GamePro Master.

## Local Setup

1. Install the latest Shopify CLI.
2. Clone the repository.
3. Run `shopify theme dev --store=your-store.myshopify.com`.
4. Run `shopify theme check` before submitting a pull request.

## Guidelines

- Keep storefront content server-rendered with Liquid where practical.
- Use JavaScript as progressive enhancement.
- Keep sections merchant-configurable and responsive.
- Add translation keys for customer-facing text.
- Avoid committing store credentials, private customer data, or assets you cannot redistribute.
- Keep pull requests focused and explain visible behavior changes.

## Pull Requests

Include a short summary, verification steps, and screenshots for visual changes.
